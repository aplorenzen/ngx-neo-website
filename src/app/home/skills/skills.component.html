Key Skills

To give you some idea about what I do best, I thought I would supply a list of what I consider my essential skills.

Docker

Since 2014

I am quite seasoned here, having been on board since before the 1.0 release. And since 2016, I have been advocating using Docker in production. These days I basically wrap everything I do in Docker containers - it's a great time to be a developer.

Bash

Since 2010

I find myself loving the linux shell more and more with every passing year. With this in my belt, I am able to automate complex chains of tools and events. The unix philosophy definitely strikes a chord with me.

Java

Since 2000

There's a lot of new languages landing in the development space these day, and a lot of them have valid use cases and are worth looking into. And still, java is in my opinion managing to stay current. With it's age old proposition of solving pretty much everything for you, pretty much everywhere.

Linux

Since 2010

I started exploring Linux back in 2010, and it's been something of a romance ever since. I was raised on DOS and Windows, and was a bit intimidated by Linux at first. But it did not take long for me to awaken to the advantages of Linux and the immense universe of distros, tools and knowledge that exists.

Angular

Since 2016

As many others with an educational background in programming, I started as a backend programmer. Only in recent years have I also picked up the frontend discipline, in my own time. When Angular 2 was released, I realized that typescript would be a great path for me, given my background. And it has been, I really enjoy Angular and the npm universe.

Spring Boot

Since 2015

Spring, is a big framework - really big. And tying a lot of it together is Spring Boot. With this, I can set up self contained and scalable micro services and websites, in just about no time. And it just plays so well with Docker - it's a match made in silicon heaven.

Since 2014
